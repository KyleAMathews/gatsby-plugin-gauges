# gatsby-plugin-gauges

Easily add [Gauges analytics](https://get.gaug.es) to your Gatsby site.

## Install

`npm install gatsby-plugin-gauges`

## How to use

```js
// In your gatsby-config.js
plugins: [
  {
    resolve: `gatsby-plugin-gauges`,
    options: {
      siteId: 'YOUR_GAUGES_SITE_ID',
    },
  },
]
```
